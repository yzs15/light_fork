# light_fork

Provide a low memory footprint service

through fork and dump anonymous memory area to file

the service load data from file to memory on demand

## installation

1. lightfork
```bash
cd ipdos/build/
bash build.sh
```

2. python extension
```bash
cd light_fork
bash build.sh
```

3. ruby extension
```bash
cd light_fork_ruby
bash build.sh
```

cd ../../
